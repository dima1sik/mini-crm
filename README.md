# MiniCRM

MiniCRM is a simple CRM application for managing clients, sales deals, follow-up tasks, reports and system users.

## Features

- Dashboard with sales KPI
- Client list and Client 360 view
- Sales pipeline with deal stages
- Follow-up tasks
- Sales reports
- User management
- Role-based access
- Local data persistence

## How to Run

dotnet restore
dotnet run
