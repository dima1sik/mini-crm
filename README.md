# MiniCRM

MiniCRM is a small C# / .NET CRM web application for managing clients, sales deals, follow-up tasks, reports, users and role-based access.

## Features

- Dashboard with sales KPI cards
- Client list and Client 360 view
- Deal pipeline with stages and history
- Follow-up task tracking
- Reports with filters
- User management
- Role-based access
- Local data persistence

## Tech Stack

- C#
- .NET
- ASP.NET Core
- HTML
- CSS
- JSON

## How to Run

dotnet restore
dotnet run

Open the local URL shown in the terminal.
