# MiniCRM

MiniCRM is a C# web application for managing clients, sales deals, follow-up tasks, reports and user access.

## Overview

The project demonstrates a small CRM system focused on common sales and business workflows. It includes client management, sales pipeline tracking, task follow-ups, reporting, user administration and role-based module access.

## Features

- Dashboard with sales KPI cards
- Client list and Client 360 view
- Client notes and related deals
- Deal pipeline with stages and history
- Follow-up task tracking
- Reports with filters and saved views
- PDF and XLS report export
- User management
- Role-based access to modules
- Account blocking and unblocking
- Audit log for admin actions
- Local data persistence

## Tech Stack

- C#
- .NET
- ASP.NET Core
- HTML
- CSS
- JSON

## How to Run

1. Open the project folder.
2. Restore project dependencies: dotnet restore.
3. Run the application: dotnet run.
4. Open the local URL shown in the terminal.

## Notes

This is an educational CRM project focused on business workflows, user interface structure and role-based access control.
